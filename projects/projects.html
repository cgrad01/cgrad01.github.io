Juke December 2015

Juke is an app built using the Spotify API, that aims to create a collaborative playlist creation. Juke allows users to upload their playlists, and then invite their friends to up and down vote songs, and suggest new songs. The host user then has the ability to reorder the playlist by votecount, and append new songs from the list of friends' suggestions. View the Github Repository Here
