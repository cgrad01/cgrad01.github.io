Looping: JavaScript vs. Ruby

Ruby

"Each" enumberable

Syntax:

array = [1,2,3,4,5]

array.each do |item|

[do code to each item once]

end

"For/in" loops

Syntax:

array = [1,2,3,4,5]

for i in array

[do this code to each member of the array once, stop at the end]

end

"While" loops

Syntax:

while [conditional] do

[do this code]

end

JavaScript

"For" loops

Syntax:

for (var counter=[start]; counter < [end]; counter[increment]) {

[do this code];

}

"For/in" loops

Syntax:

var array = [1,2,3,4,5]

for (var numbers in array) {

[do this code to each member of the array once, stop at the end];

}

"While" loops

Syntax:

while ([condition]) {

[do this code];

}

"Do/while" loops

Syntax:

do {

[do this code];

} while ([conditional]);
